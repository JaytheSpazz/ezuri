# Ezuri
A Simple Linux ELF Runtime Crypter.
An unpacker by [f0wl](https://github.com/f0wl) can be found at https://github.com/f0wl/ezuri_unpack.

# Usage
Clone this repo, build with `go build -o ezuri .` and run `./ezuri` executable.
